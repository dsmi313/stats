# R

This folder contains R code and analysis scripts for the `stats` repository.

Organize code by part and section:

- `R/PartI`: binomial simulation, bootstrap, and EASE building blocks.
- `R/PartII`: Delomas & Hess likelihood and composition estimation.
- `R/PartIII`: SCRAPI internals and real data audit.
- `R/PartIV`: parallel implementations and package comparisons.
- `R/PartV`: broader fisheries package examples and external perspectives.
- `R/PartVI`: Bayesian sidebar, JAGS workflows, and STADEM.
- `R/PartVII`: synthesis memo and summary diagnostics.
- `R/PartVIII`: integrated Bayesian routing and composition model.
